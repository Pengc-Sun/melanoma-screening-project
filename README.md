# AI-Assisted Mobile Melanoma Screening Project

This repository contains the project log-book, presentation materials and selected experimental evidence for an AI-assisted mobile melanoma screening system based on multi-modal fusion.

## Experimental Evidence

- [Experiment 1 — Image-only ResNet50 baseline](experiments/baseline)
- [Experiment 2 — Multi-modal fusion model](experiments/multimodal)
- [Experiment 3 — Application integration test](experiments/application-integration)

The uploaded model evidence includes metric summaries, training histories, anonymised prediction outputs and evaluation figures. The ISIC2020 dataset and trained model checkpoints are not included because of dataset distribution requirements and file size.

## Medical Disclaimer

This project is intended for research and educational purposes. Its predictions must not be treated as a final medical diagnosis.
